### Project setup

client:

client ディレクトリに移動

```
$cd client
```

以下のコマンドを実行してパッケージをインストール

```
$npm install
```

or

```
$yarn install
```

以下のコマンドで開発サーバーを起動

```
$npm start
```

of

```
$yarn start
```
